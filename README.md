## Clone this project

```bash
> git clone https://github.com/mcpr259/botmcpr/
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> npm i
```

### Usage
Before running this script, first edit [`THIS`](https://github.com/mcpr259/botmcpr/blob/master/msgHndlr.js#L55) section with your WhatsApp number, remember your WhatsApp number!  Not a bot number, then
```bash
> npm start
```

### Terms and conditions
Your texts and your whatsapp username will be stored on our servers as long as the bot is active, your data will be erased when the bot goes offline.
